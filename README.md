# 대한항공 채용 준비 사이트

> FlyReady Lab 제공 | 2025년 대한항공 객실승무원 채용 대비

## 기능

1. **🎤 모의면접**: 대한항공 기출 질문 기반 AI 모의면접
2. **📝 자소서첨삭**: 대한항공 자소서 문항별 AI 첨삭
3. **📰 기업분석&뉴스**: 대한항공 최신 정보 + 면접 필수 숙지 뉴스
4. **💡 면접꿀팁**: 복장, 메이크업, 면접 유형별 가이드
5. **😊 표정분석연습**: 웹캠 기반 실시간 표정 분석
6. **💬 AI챗봇**: 대한항공 채용 전문 AI 상담

## 설치

```bash
pip install -r requirements.txt
```

## 실행

```bash
streamlit run app.py
```

## 환경 변수

Streamlit Cloud 배포 시 Secrets에 설정:

```toml
ANTHROPIC_API_KEY = "sk-ant-xxxxx"
```

## 배포

1. GitHub에 push
2. Streamlit Cloud 연결
3. Secrets 설정
4. 배포 완료

## 면책 고지

본 서비스는 대한항공 공식 서비스가 아닙니다.
모든 정보는 공식 출처 기반이며, 최신 정보는 대한항공 공식 채용 페이지에서 확인하세요.
